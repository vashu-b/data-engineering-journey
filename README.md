# 📊 Customer Data Pipeline Project

## 🚀 Overview

This project demonstrates an end-to-end data pipeline built using Python, Pandas, and MySQL.

The pipeline extracts raw data from a CSV file, transforms it using Python, and loads it into a structured MySQL database for analysis.

---

## 🧠 Problem Statement

Raw data is often unstructured and not directly usable for analysis.
This project solves that by building a pipeline to clean, transform, and store data efficiently.

---

## ⚙️ Tech Stack

* Python
* Pandas
* MySQL
* SQL
* Git & GitHub

---

## 🔄 Pipeline Flow

CSV → Pandas → Transformation → MySQL

---

## 🛠️ Steps Involved

### 1. Data Ingestion

* Read CSV file using Pandas

### 2. Data Transformation

* Applied revenue transformation (+10%)
* Cleaned and structured dataset

### 3. Data Loading

* Connected Python to MySQL
* Inserted processed data into database

---

* Worked with real-world data formats (CSV)
* Integrated Python with MySQL database
